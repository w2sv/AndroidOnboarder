## Installation with gradle & jitpack

```gradle
implementation 'com.github.w2sv:onboarding:x.y.z'
```
or get the latest snapshot of a certain branch:
```gradle
implementation 'com.github.w2sv:onboarding:<BRANCH-NAME>-SNAPSHOT'
```
